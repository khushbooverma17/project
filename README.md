## Project Description
**Food Secure** is a decentralized platform designed to promote transparency and trust in food security initiatives.  
By leveraging blockchain technology, it allows users to donate directly to verified food distribution programs and track how resources are allocated.

## Project Vision
To create a transparent, community-driven ecosystem that ensures food donations reach those in need efficiently and responsibly.  
Food Secure empowers donors, NGOs, and communities through verifiable data on food distribution and fund utilization.
